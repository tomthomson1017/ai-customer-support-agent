# AI Customer Support Agent Builder

An AI-powered customer support agent builder using OpenAI API and Streamlit.

## What It Does
- Custom company setup through a simple UI
- AI answers questions based only on your company data
- Logs unanswerable questions for human follow-up
- Export chat history as a text file

## Tech Stack
- Python, OpenAI API, Streamlit

## How to Run
1. pip install streamlit openai
2. export OPENAI_API_KEY="your-key"
3. streamlit run app.py
